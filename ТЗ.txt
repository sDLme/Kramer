Всем привет. Добавил notes к desktop версии макета Kramer.
ТЗ:

- Использовать bootstrap grid ( но с умом )

- Проанализировать макет для более чистого и “умного” кода. Например одинаковые оступы сверху и снизу в каждой секции, одинаковые заголовки по шрифту и сайзу и так далее

- Section Welcome - Секция на высоту экрана. Background-ом сделать video, которое само проигрывается и по окончанию начинает проигрываться сначала ( loop ). На мобильном разрешении background поменять с video на img для более быстрой работы, так как video может долго грузится на моб инете

- Section About -  сделать в нем sticky header ( этот слой надо включить, по дефолту он выключен ). Изначально мы не видим хедер на секции about пока не доскролим эту секцию в верхнюю точку экрана. После того как about section доскролили до верхней точки экрана - появляется sticky header и он всегда будет fixed вверху экрана. Он пропадает когда мы скролим назад на секци welcome. 
Для этого решения посмотрите bootstrap afix!
На мобильном разрешении sticky header убираем, так как верхний хедер будет фиксед всегда

- Portfolio section. “all design web illustration motion” - это фильтры по которым отображается список ниже. Его нужно сделать. Для примера использовать izotope либу (ссылку дал в авокоде) . Это задание optional. 
Кто не будет делать фильтрами, тогда список вышеупомянутый это список ссылок, по active и hover применяется bg зеленого цвета
Список сделать с помощью bootstrap grid
